# codelintcli
## 介绍
一个脚手架，封装了alloy eslint，prettier的配置，支持Vue和React，稍后会支持Typescript。
可选择安装与Vue或者React相关的eslint和prettier(主要是alloy的规范)的内容，同时还有editorConfig，lint-staged，husky的内容，并生成默认配置，同时自动安装相应的包。
## 安装
```
#npm
npm i codelintcli -g

#yarn
yarn add codelintcli global
```
## 使用
```
codelintcli i
```
## 帮助
```
codelintcli -h
```
